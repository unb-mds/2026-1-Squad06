# Guard.IA

Sistema de monitoramento legislativo sobre proteção de crianças e adolescentes na internet.

## Sobre o projeto

O Guard.IA tem como objetivo acompanhar proposições legislativas relacionadas à proteção de crianças e adolescentes no ambiente digital.

## Funcionalidades

- Coleta de proposições legislativas
- Filtragem por palavras-chave
- Classificação por temas
- Armazenamento dos dados
- Visualização em dashboard
