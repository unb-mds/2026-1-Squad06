## 📖 Descrição

Este documento apresenta o estudo dos fundamentos do Figma, ferramenta amplamente utilizada para criação de interfaces, prototipação e organização visual de aplicações.

O objetivo é compreender seus principais recursos e sua aplicação prática no desenvolvimento de sistemas, especialmente no planejamento de interfaces antes da implementação em código.

## 💻 Aplicação no Projeto

No projeto de monitoramento legislativo sobre proteção de crianças na internet, o Figma pode ser utilizado para:

- Estruturar o dashboard de análise legislativa  
- Criar telas de busca e filtragem de proposições  
- Organizar a visualização de dados classificados via NLP  
- Definir a experiência do usuário (UX) da plataforma  
- Planejar a interface antes da implementação em código  

O uso do Figma permite transformar dados complexos em interfaces claras, intuitivas e funcionais.
