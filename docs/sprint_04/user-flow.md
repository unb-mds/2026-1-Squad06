# 🔁 Criação do User Flow do sistema

## 📖 Descrição
Foi desenvolvido o user flow do sistema com o objetivo de representar o comportamento do usuário durante a navegação na aplicação.

O fluxo foi construído com base no protótipo existente, garantindo coerência entre as telas e as interações disponíveis.

---

## 🎯 Objetivo
Mapear o caminho do usuário dentro do sistema, desde o acesso inicial até a visualização de informações detalhadas.

---

## 🧩 Entregas realizadas

- ✔ Definição do fluxo principal do usuário
- ✔ Representação da navegação entre páginas (Dashboard, Proposições, Ranking e Mapa)
- ✔ Inclusão das etapas de login/cadastro
- ✔ Estruturação das interações (filtros, busca e visualização de dados)

## 💡 Observações
O fluxo considera:

- Entrada do usuário no sistema
- Processo de autenticação (login/cadastro)
- Navegação entre as principais páginas
- Aplicação de filtros e busca
- Visualização de dados e detalhes

---

## ✅ Resultado esperado
Facilitar o entendimento do funcionamento do sistema e servir como base para o desenvolvimento e organização das funcionalidades.
