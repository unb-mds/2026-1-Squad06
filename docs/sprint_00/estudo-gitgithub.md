Qual é a DIFERENÇA entre Git e GitHub?
- Git = Controla versões do seu projeto
- GitHub = Onde você deixa e compartilha seu projeto
- É indicado aprender nesta ordem:

1. Básico do Git:


- git init
- git status
- git add
- git commit
- git log

2. GitHub:

- git clone
- git push
- git pull
- Quando o básico já estiver claro:

3. Branches.

git branch
git switch
git merge

Por último:
4. Colaboração.

- Pull Request
- Conflitos de Merge
- Fork

5. O melhor jeito de Estudar é PRATICANDO

5.1. Criar uma pasta
5.2. Iniciar Git
5.3. Criar/Editar arquivos
5.4. Fazer commits
5.5. Subir para o GitHub
5.6. Criar uma branch e fazer um merge

6. Priorize:

- Entender o ciclo editar -> add -> commit
- Aprender a evitar a baixar o código do GitHub
- Saber criar e juntar branches
