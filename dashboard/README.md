# Dashboard de Visualização

Esta pasta contém a primeira versão da etapa de visualização do projeto de monitoramento legislativo.

## Tecnologias
- Streamlit
- Plotly
- Pandas

## Arquivos
- `app.py`: aplicação principal

```bash
pip install streamlit plotly pandas
